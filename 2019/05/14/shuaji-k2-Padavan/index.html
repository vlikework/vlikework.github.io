刷机之斐讯路由器K2尝试华硕Padavan固件
发布于 2019-05-14 / 更新于 2019-05-14 / 路由器刷机 / 热度 0 / 喜欢 0

为什么是斐讯K2(PSG1218)路由器？ 斐讯K2路由器的0元购活动，399元购买–>K2序列号绑定联币理财账户–>1月后返回399元+3元利息->提现到银行卡。
当然，现在对于“斐迅”的“零元购”只能呵呵了（因为联币金融已经跑路啦！！）。

斐讯路由器K2尝试华硕Padavan固件，本次分两种情况介绍刷固件的过程：

  1. 联网情况下的一键刷固件方法
  2. 不联网下的手工刷固件方法

第一种一键刷路由固件的方式，我尝试的时候出现了404页面，可能已经不能使用这种方式了，因此第二种方式就有了用武之地了。

K2路由器刷固件的步骤说明

第一种，具备联外网的情况

1，首先保证当前使用的K2可以正常上网。
2，使用网线，将路由器与电脑连接。
3，拔掉K2电源。
4，按住Reset键，插上电源，10秒后松开Reset键。
5，访问斐讯默认后台地址192.168.2.1，默认账户密码admin。
6，开始刷机，有两种方式，一个标点符号都不要少，否则变砖完蛋：

方式1：只刷入Breed，在Chrome地址栏打开下面的链接：

http://192.168.2.1/goform/gra_NTPSyncWithLocal?text_year="2016|`if wget -O /tmp/breed.bin http://breed.hackpascal.net/breed-mt7620-phicomm-psg1208.bin;then mtd_write write /tmp/breed.bin Bootloader;echo breed ok! >/etc_ro/web/index.asp;fi`"

方式2：同时刷入Breed和华硕Padavan固件，在Chrome地址栏打开下面的链接：

http://192.168.2.1/goform/gra_NTPSyncWithLocal?text_year="2016|`if wget -O /tmp/asus.sh http://code.taobao.org/svn/asus_bin/asus.sh;then sh /tmp/asus.sh;fi`"

7，等待一分钟，一定不要关闭，等待，等待。
8，出现以下画面，表示已经成功。此界面可能会跳转，可返回后，迅速点击 click here那句话。
9，重启路由，不行断电20秒再通电。shaoit.com
10，访问华硕Padavan默认地址192.168.123.1，输入默认账户密码admin，即可管理路由器。

第二种，不连接网络下刷路由器固件操作情况

其实离线就是提前将需要的固件下载到本地，然后再执行刷固件操作

  1. 下载 breed
  2. 下载 华硕Padavan固件，找到最新版本的RT-AC54U-GPIO-1-PSG1208-64M_3.4.3.9-099.trx文件下载下来。
  3. 下载华硕Kernel文件 。
  4. 下载 [斐讯降级包]，链接: https://pan.baidu.com/s/1geGtOTd 密码: u4rg

准备好了上述固件包，就可以开始刷了。

降级操作步骤。

总结起来就是通过正常的升级流程把指定的低版本斐讯K2固件刷进去。

  1. 插上路由器电源，用网线将电脑与路由器的LAN口连接起来。
  2. 打开cmd命令窗口，执行ipconfig检查自己的IP地址，例如说是192.168.2.9，那么路由器地址为192.168.2.1。
  3. 打开chrome浏览器，访问http://192.168.2.1地址，进行简单的快速配置就可以（此配置无所谓的），配置完毕后找到系统管理中的手动升级页面，点击浏览按钮，选择下载的斐讯降级包 文件，然后点击升级就可以了，此时就需要等待2-3分钟。
  4. 升级成功后，就可以看到最底下的一行显示的版本信息了。

刷breed固件

具体就是将breed固件上传到路由器上，再手工刷Bootloader
其中需要说的是，不需要开telnet服务，ssh服务是可用的，用户名和密码都是root。

  1. 使用secureCRT之类的SSH客户端软件访问路由器的地址(目前这里应该是192.168.2.1) ，配置用户名密码都是root。
  2. 连接上路由器后，将本地的breed固件上传到/tmp目录下，比如文件名叫做breed-mt7620-phicomm-psg1208.bin。
  3. 上传完毕后，开始刷固件，执行的命令为mtd write /tmp/breed-mt7620-phicomm-psg1208.bin Bootloader ，这里需要说民下之前的一键刷固件使用的命令是mtd_write，而我后期实测并没有这个命令了，而有mtd命令就可以了，当然如果执行mtd命令报错，那可以尝试使用mtd_write write /tmp/breed-mt7620-phicomm-psg1208.bin Bootloader ，这个可能跟具体的斐讯的路由器底层系统版本有关。
  4. 执行大概不到1分钟就完成了。之后可以重启到breed的界面，具体操作是这样的，拔掉电源->按住Reset按钮同时插上电源，按住Reset10秒左右就可以松手了。
  5. 这时再查看下自己的ip地址ipconfig，例如是192.168.1.9，那么同理路由器地址就是192.168.1.1，通过chrome浏览器访问http://192.168.1.1页面了。

刷华硕Padavan固件

两种方法，第一种，通过浏览器页面使用的是华硕Padavan固件，文件名应该是”RT-AC54U-GPIO-1-PSG1208-64M_3.4.3.9-099.trx”;第二种，使用命令行方式刷Kernel,使用的文件名为”asus_bin”。

方法1：通过浏览器页面操作方式

  1. 通过chrome浏览器访问http://192.168.1.1页面，找到固件管理中的固件升级，上传下载的华硕Padavan固件，点击开始升级。
  2. 升级成功后会提示刷新成功，之后重启路由器就可以看到新的路由器管理页面了。

方法2：通过命令操作方式

  1. 与刷breed方式相同，先通过ssh方式将华硕Kernel文件上传到路由器的/tmp目录下，比如文件名为asus_bin。
  2. 然后执行mtd write /tmp/asus_bin Kernel命令开始刷kernel。
  3. 刷成功后，重启路由器就可以看到新的路由器管理页面了。

最后，刷新成功后，登录的地是就是192.168.123.1，默认的用户名和密码都是admin 。

最后

刷机有风险，以上总结为本人亲测可行，祝你刷机成功！若你在刷机过程中遇到了什么问题也可以与我交流，希望可以帮到你愉快的玩转K2路由器。
