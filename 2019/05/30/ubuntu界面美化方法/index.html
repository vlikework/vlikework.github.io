Ubuntu 18.04 LTS 界面美化方法
发布于 2019-05-30 / 更新于 2019-05-30 / Linux Ubuntu 系统优化 / 热度 0 / 喜欢 0

Ubuntu 系统是一款非常入门级的Linux操作系统， 简单易用 ， 软件源也比较丰富， 大多数软件工具可以直接下载安装使用，对于不会编程的小白来说，省去了不少折腾时间。
当我们第一次安装Ubuntu后其实就可以使用系统自带的软件了，不过自带软件并不多，所以常常需要我们自己安装一些需要的软件，这很容易理解，就是按照自己的功能需要进行自定义安装。

常用 Ubuntu 的朋友都知道 ，每年都有两个版本发布，以2018年为例， 第一个版本在4月份发布，命名为 “Ubuntu 18.04 LTS” ,LTS 是长生命周期版本,每两年发布一个LTS版本，通常会有四年的更新维护期; 第二个版本在 10月份发布，命名为 “Ubuntu 18.10”， 这个版本是个过渡版本，只有1年的更新维护期，通常这个版本会添加一些新功能特性，稳定性和问题也会多一些。

另外，每个发布版本都会有个代号Code , 通常都是固定的形容词加上一种动物命名，历史上的各个版本命名如下:

图片来自 维基百科

界面美化操作

安装 gnome-tweat-tools

sudo apt-get install -y gnome-tweak-tool

推荐安装插件

sudo apt install -y gnome-shell-extension-dashtodock gnome-shell gnome-shell-extensions gnome-shell-extension-top-icons-plus

安装主题

mkdir ~/.themes ~/.icons
cd ~/apps/github/ && mkdir gui
cd gui

## 主题下载
git clone https://github.com/vinceliuice/vimix-gtk-themes.git

cd ./vimix-gtk-themes && ./Install


## 图标下载
git clone https://github.com/vinceliuice/vimix-icon-theme.git
cd ./vimix-icon-theme && ./install.sh

安装 cairo-dock (自己可选，如果喜欢自带的 dock 就不用安装了)

sudo apt-get install cairo-dock

常用工具安装

安装 Git / Vim / zsh / oh-my-zsh

mkdir ~/apps/
mkdir ~/apps/{github,bin}

## 安装 Git ， vim , zsh , curl 工具命令
sudo apt install -y git vim zsh curl


## 安装 oh-my-zsh ,让Terminal终端显示更友好
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

## 修改 oh-my-zsh 主题为 agnoster

gedit ~/.zshrc

ZSH_THEME="agnoster"


## 安装 powerline 字体
cd ~/apps/github
git clone https://github.com/powerline/fonts.git
cd fonts && ./install.sh

设置`Terminal`终端的字体为"Droid Sans Mono for Powerline Regular" 后，重新打开终端就可以看到主题效果了，如果我们经常使用 git 进行版本管理， ohmyzsh 会极大提高你的工作效率和心情的。



## 安装 spacevim ，让 vim 操作更友好快捷， 懒人必备，当然如果你有自己的配置文档版本管理就更好了，你可以减少很多配置。详细配置介绍请访问 https://spacevim.org
curl -sLf https://spacevim.org/install.sh | bash


## 执行 vim 命令，插件会自动安装
vim

安装代理工具

ShadowsocksR代理工具安装，这款代理是目前非常强壮的代理，而且使用起来也比较简单。

v2ray代理工具功能非常强大，配置化更多，更复杂些。

sudo apt install curl
bash <(curl -L -s https://install.direct/go.sh)

proxychains4是命令行代理工具，例如你使用 curl 时，可以通过它增加代理转发功能。

sudo apt install proxychains4

安装 exfat 支持

Ubuntu下使用FAT32格式文件系统时，需要安装exfat-utils才可以正确挂载上，否则无法识别我们的U盘。

unar是一款压缩/解压工具，可以解决使用unzip解压中文压缩包时乱码问题。

sudo apt install -y exfat-utils unar

解决联想电脑开机 / 插拔耳机噪音（pop noise）

解决开关机噪音：

创建一个文件 /etc/modprobe.d/modprobe.conf ，写入下面内容保存即可：

options snd-hda-intel model=,generic

解决耳机插拔噪音：

把 alsa 的 auto mute 关掉，插拔耳机就没有噪音了:

sudo alsamixer

按 F6 选择第二个显卡 (HDA INTEL PCH)，然后把 auto mute 改为 disabled;

sudo alsactl store    # 保存配置

安装办公工具软件

截图工具： deepin-screenshot(gnome下强烈推荐) ,也推荐使用 shutter 。
录屏工具: deepin-screen-recorder
思维导图工具: XMind Zen 下载地址
Office办公软件： WPSLinux必然是首选了， 下载地址
音乐播放器 : (网易云音乐)[https://music.163.com/#/download]

sudo apt install -y  deepin-screenshot deepin-screen-recorder vlc

安装搜狗输入法

sudo apt install fcitx

然后下载搜狗输入法 Download , 下载后开始安装.

这里我下载到了 ~/下载目录了，因此我直接执行下面命令开始安装。

sudo apt install -y ~/下载/sougoupinyin_2.2.0.0108_amd64.deb

安装完fcitx和搜狗输入法后，按 开始 键，在 Dash 中找到 “语言支持” ， 将键盘输入法设置为fcitx，重启电脑。
重启后，登录系统后，点击右上角的小键盘图标，选择搜狗输入法即可使用啦， 以后切换输入法时直接按快捷键 Ctrl Space(Ctrl键和空格键的组合)。

安装 电报聊天软件TG

先下载TG 压缩包 ， 下载链接


## 下载后开始安装, 解压软件命令到 /opt 目录下 （要 root 权限的）
$ sudo tar zxvf ./tsetup.1.7.0.tar.xz -C /opt

## 添加Ubuntu 启动桌面快捷方式

$ sudo vi /usr/share/applications/telegram.desktop
